# Multiclass-Classification-with-PyTorch
Multiclass classification with PyTorch using the Iris dataset, plus an exercise on Fetal Health Classification. Covers data preprocessing, linear model with Softmax, Cross-Entropy Loss, SGD training, and evaluation.

whats inside : 
Load and explore the Iris dataset 
- Train/validation/test split + standardisation
- Linear model with `Softmax` activation
- Training with `CrossEntropyLoss` and SGD
- Monitor loss & accuracy curves
- Test evaluation (100% accuracy on Iris)
